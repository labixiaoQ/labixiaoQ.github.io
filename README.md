# Academic Homepage

A minimalist academic personal homepage, prepared for GitHub Pages.

## Personalize

Update the placeholders in `app/page.tsx` for your name, role, biography,
research interests, publications, projects, city, and email address. Replace
`public/og.png` with a new social preview image if your name changes.

## Publish

Create a GitHub repository named `labixiaoQ.github.io`, push this project to
its `main` branch, then select **GitHub Actions** under **Settings → Pages**.
The included workflow builds and deploys the site automatically.
